How to open the terminal (if it is not open already):
- On the top bar, next to Run, click on Terminal, and click new Terminal.

How to start the web server:
- In a terminal, type `hugo server` without the quotes, and press enter

How to access the website:
- Once the web server is started, open firefox (or google chrome) and go to http://localhost:1313

Every change you make are applied live once the server is started but sometime you'll have to stop and restart the whole server.
To stop the server:
- In the terminal press CTRL + C.
